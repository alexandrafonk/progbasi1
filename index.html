¿Qué son las etiquetas semánticas?

Las etiquetas semánticas en HTML sirven para definir una estructura clara de mi sitio web.

Tabla de etiquetas semánticas

ETIQUETA DESCRIPCIÓN
<header> sección en la parte superior de una página web que contiene elementos clave como el logo, la navegación principal y el eslogan, que se repite en todas las páginas para ofrecer una marca e identidad consistentes y facilitar la navegación al usuario
<nav> se utiliza para agrupar enlaces que sirven como navegación principal dentro de un sitio web o en una sección específica de la página.
<main> Es el cuerpo principal del texto o contenido del diseño de un sitio web.
<footer> IMG para inserta imagnes
<section> a para crear hipervinculos
<aside> Incorporará texto que no pertenece a las secciones principales de tu sitio web, pero que debería aparecer en un área o sección establecida.
<article> Es útil para reutilizarse y también se considera distribuible de forma independiente
