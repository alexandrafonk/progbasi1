Estos son solo algunos selectores básicos

Los selectores de HTML son patrones que se utilizan para seleccionar y aplicar estilos a elementos específicos dentro de un documento HTML. Aquí hay un resumen de los tipos más comunes de selectores:

Selector de etiqueta

Selecciona todos los elementos con una etiqueta HTML específica, por ejemplo, p seleccionará todos los párrafos en el documento.

Selector de clase:

Selecciona elementos que tienen un atributo class específico, por ejemplo, .clase seleccionará todos los elementos con la clase clase.

Selector de ID:

Selecciona un elemento con un atributo id específico, por ejemplo, #identificador seleccionará el elemento con el ID identificador.

Selector descendente:

Permite seleccionar un elemento dentro de otro elemento, por ejemplo, p span seleccionará todos los spans que están dentro de párrafos.

Selector universal:

Selecciona todos los elementos en la página, representado con el asterisco *.
